### 📝 เกี่ยวกับโปรเจกต์ (About The Project)

โปรเจกต์นี้เป็นระบบ **การจองโต๊ะร้านอาหาร** ที่พัฒนาขึ้นเพื่อแก้ปัญหาการจองโต๊ะที่ซับซ้อน ช่วยให้ลูกค้าสามารถตรวจสอบสถานะโต๊ะว่างและทำการจองได้ทันที ผ่านหน้าจอที่ใช้งานง่าย พร้อมทั้งมีระบบหลังบ้านสำหรับผู้ดูแลร้านในการจัดการข้อมูลการจอง

### 🚀 ฟีเจอร์หลัก (Key Features)

* ✅ **ระบบค้นหาและจอง:** ลูกค้าสามารถดูผังร้านและเลือกโต๊ะที่ว่างได้แบบ Real-time
* ✅ **จัดการสถานะ:** แสดงสถานะโต๊ะ (ว่าง/จองแล้ว/กำลังใช้งาน)
* ✅ **ประวัติการจอง:** ลูกค้าสามารถดูประวัติการจองย้อนหลังได้
* ✅ **Admin Dashboard:** ผู้ดูแลระบบสามารถ ยืนยัน หรือ ยกเลิก การจองได้
